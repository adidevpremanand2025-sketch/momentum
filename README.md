# Momentum OS

A single-file personal productivity dashboard built with vanilla HTML, CSS, and JavaScript.

## Features

- **Customizable dashboard** — drag-and-drop widgets, resize, reorder
- **AI Coach** — powered by Claude (Anthropic API), personalized to your fitness and nutrition data
- **Fitness tracker** — workout logging, volume targets, Pomodoro timer, strength trend chart
- **Nutrition tracker** — macro breakdown, live meal logging, daily targets
- **Smart planner** — timeline view, calendar sync toggles
- **Builder** — custom trackers, theme colors, layout modes (balanced / dense / focus)
- **Dark/light mode** — persisted to localStorage

## Tech

Vanilla JS · CSS custom properties · Anthropic Messages API · No build step · No dependencies

## Run it

Download `momentum-os.html` and open it in a browser. That's it.

To enable the AI Coach, add your Anthropic API key — see [Getting an API key](https://console.anthropic.com/).
